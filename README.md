# 🎓 MED2019 - Portal Acadêmico Integrado

Este repositório armazena o protótipo de interface de um **Portal Acadêmico e Plataforma de Estudos EAD** voltado para estudantes de vestibulares de Medicina. O projeto foi desenvolvido utilizando padrões de semântica estrutural em **HTML5** e estilização avançada com **CSS3**.

---

## 🚀 Arquitetura & Interface Implementada

* **Estrutura Semântica (HTML5):** Utilização correta de tags estruturais (`header`, `div`, `span`) organizando fluxos lógicos de cabeçalhos de marca e menus navegacionais.
* **Layout Fluido & Flexbox:** Implementação do **CSS Flexbox** para alinhamento em cruz bidimensional, garantindo que os cards de ciclos interativos se alinhem horizontalmente com quebra de linha responsiva (*flex-wrap*).
* **Componentização Visual:** Criação de cápsulas interativas customizadas com círculos internos simétricos centralizados verticalmente para a seleção de Ciclos de Aprendizado por semana.

---

## 🛠️ Tecnologias Utilizadas

* **Estruturação:** HTML5 Semântico
* **Estilização:** CSS3 (Flexbox Layout & Efeitos de Sombra/Text-Shadow)
* **Design:** Conceito de Dashboard EAD Minimalista (Padrão Dark Mode Superior)

---

## 💻 Como Executar o Projeto Localmente

Por se tratar de uma aplicação front-end estática nativa, não é necessária a instalação de gerenciadores de pacotes externos ou servidores Node.js.

### 1. Clonar o repositório
```bash
git clone https://github.com
cd HTML-CSS
```

### 2. Executar a Interface
Basta dar um duplo clique no arquivo **`index.html`** ou abri-lo diretamente através de qualquer navegador de internet de sua preferência (Google Chrome, Firefox, Edge, etc.).
